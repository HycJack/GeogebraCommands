LUDecomposition Command

CAS Syntax

LUDecomposition( <Matrix> )

Calculates the LU decomposition of the given matrix.

LUDecomposition({{2,0},{1,1}}) returns the matrices \(\begin{pmatrix}0&1\\1&0\end{pmatrix}\),\(\begin{pmatrix}1&0\\2&1\end{pmatrix}\) and \(\begin{pmatrix}1&1\\0&-2\end{pmatrix}\).

See also QRDecomposition command.
