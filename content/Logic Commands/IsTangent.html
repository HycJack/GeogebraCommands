IsTangent Command

IsTangent( <Line>, <Conic> )

Decides if the line is tangent to the conic.

Normally this command computes the result numerically. This behavior can be changed by using the Prove command.

IsTangent(Line((0,0),(1,0)),Circle((0,1),1)) yields true.

See also AreCollinear, AreConcurrent, AreCongruent, AreConcyclic, AreEqual, AreParallel, ArePerpendicular commands.
